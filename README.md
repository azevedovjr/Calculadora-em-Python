# Calculadora-em-Python

Aplicação desenvolvida em Python com menu interativo para realizar operações matemáticas básicas como soma, subtração, multiplicação e divisão. O projeto utiliza estruturas condicionais e de repetição para criar uma experiência simples de uso no terminal.

Tecnologias utilizadas:

Python

Lógica de programação

Estruturas condicionais (if, elif, else)

Estruturas de repetição (while)
